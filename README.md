# Evaluación JAVA - EY

### Tecnologías utilizadas

* Java 11+
* Spring Boot 3.1.4
* Hibernate
* Swagger
* H2 (Database)


### Realizar pruebas

Una vez levantada la aplicación, se puede ingresar a:
* http://localhost:8080/swagger-ui/index.html (Probar aplicación)
* http://localhost:8080/h2-ui/ (Base de Datos)
